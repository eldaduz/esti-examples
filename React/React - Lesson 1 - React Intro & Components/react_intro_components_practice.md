# תרגול React — שיעור 1: Intro & Components

> תרגול זה מבוסס על שיעור הפתיחה בריאקט: JSX, סוגריים מסולסלים `{}`, קומפוננטות, `export default`, `import`, ושימוש חוזר ברכיבים.

---

## הנחיות כלליות

בכל תרגיל:

1. כתבו את הקוד בתוך פרויקט React קיים.
2. השתמשו בקבצי `.jsx` כאשר אתם כותבים קומפוננטות.
3. שמות של קומפוננטות חייבים להתחיל באות גדולה, למשל: `Title`, `UserCard`, `ShoppingList`.
4. אם יצרתם קומפוננטה בקובץ נפרד — אל תשכחו לעשות לה `export default`.
5. אם אתם משתמשים בקומפוננטה בתוך `App.jsx` — אל תשכחו לעשות לה `import`.

---

## תרגיל 21 — מצאו ותקנו את הטעות

```jsx
import Title from "./Title";
```

הקוד לא עובד. תקנו את הבעיה.

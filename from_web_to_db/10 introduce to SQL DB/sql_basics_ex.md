# 🧱 שלב 1 – הוספת טבלה חדשה

## 📌 מה עושים?

הריצו את הקוד הבא ב־SQL Editor כדי ליצור טבלה חדשה בשם employees ולהכניס אליה נתונים:

```sql
-- =========================
-- Employees Table
-- =========================

DROP TABLE IF EXISTS employees;

CREATE TABLE employees (
  id SERIAL PRIMARY KEY,
  first_name VARCHAR(50),
  last_name VARCHAR(50),
  department VARCHAR(50),
  salary INT,
  hire_date DATE,
  is_active BOOLEAN
);
```

# 💡 טיפים חשובים

✔ SQL לא רגיש לאותיות גדולות/קטנות
✔ כל שאילתה מסתיימת ב־ ;
✔ סדר חשוב:

```sql
SELECT
FROM
WHERE
ORDER BY
```

# 1. הדפסת מספר אחד ואז עצירה

כתבו פונקציה רקורסיבית בשם `printOnce` שמקבלת מספר `n`.

אם `n` שווה ל־0 — הפונקציה עוצרת.
אחרת — היא מדפיסה את `n`, ואז קוראת לעצמה עם `0`.

### דוגמה

```js
printOnce(5);
```

פלט:

```js
5;
```

המטרה כאן היא רק להבין את המבנה:

```js
if (...) {
  return;
}

recursiveFunction(...);
```

# 2. הדפסה מ־n עד 1

כתבו פונקציה רקורסיבית בשם `printDown` שמקבלת מספר חיובי `n` ומדפיסה את כל המספרים מ־`n` עד `1`.

### דוגמה

```js
printDown(4);
```

פלט:

```js
4;
3;
2;
1;
```

רמז:

אם `n === 0` — עוצרים.
אחרת — מדפיסים את `n` וקוראים שוב עם `n - 1`.

# 3. הדפסת המילה "Hello" כמה פעמים

כתבו פונקציה רקורסיבית בשם `printHello` שמקבלת מספר `n` ומדפיסה `"Hello"` בדיוק `n` פעמים.

### דוגמה

```js
printHello(3);
```

פלט:

```js
Hello;
Hello;
Hello;
```

רמז:

בכל קריאה מדפיסים פעם אחת, ואז קוראים שוב עם מספר קטן יותר.

# 4. בדיקה אם מספר הגיע ל־0

כתבו פונקציה רקורסיבית בשם `countToZero` שמקבלת מספר חיובי `n`.

הפונקציה מדפיסה את `n`, ואז בכל פעם מקטינה אותו ב־1, עד שמגיעה ל־0.
כאשר היא מגיעה ל־0, היא מדפיסה `"Done"`.

### דוגמה

```js
countToZero(3);
```

פלט:

```js
3;
2;
1;
Done;
```

רמז:

תנאי העצירה הוא:

```js
if (n === 0)
```

# 5. סכום פשוט מ־n עד 1

כתבו פונקציה רקורסיבית בשם `sumDown` שמקבלת מספר חיובי `n` ומחזירה את הסכום של כל המספרים מ־`n` עד `1`.

### דוגמה

```js
sumDown(3);
```

פלט:

```js
6;
```

כי:

```js
3 + 2 + 1 = 6
```

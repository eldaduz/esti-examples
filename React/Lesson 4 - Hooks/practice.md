

# תרגול Hooks ב־React

## useState

### תרגיל 1 — קל: בחירת גודל טקסט

צרו קומפוננטה בשם `FontSizeChanger`.

הקומפוננטה תציג טקסט:

```text
React is fun!
```

מתחתיו יהיו שלושה כפתורים:

```text
Small
Medium
Large
```

בלחיצה על כל כפתור, גודל הטקסט ישתנה בהתאם.

דרישות:

* השתמשו ב־`useState`.
* שמרו ב־state את גודל הטקסט הנוכחי.
* הציגו את הטקסט עם `style`.

---

### תרגיל 2 — בינוני: בחירת מוצר מועדף

צרו קומפוננטה בשם `FavoriteProduct`.

הציגו רשימה של שלושה מוצרים ככפתורים:

```text
Laptop
Phone
Headphones
```

כאשר המשתמש לוחץ על מוצר, הציגו מתחת:

```text
Your favorite product is: Laptop
```

אם עדיין לא נבחר מוצר, הציגו:

```text
No product selected yet
```

רמז: שמרו ב־state את שם המוצר שנבחר.

---

### תרגיל 3 — בינוני: טופס סיסמה עם הצגה / הסתרה

צרו קומפוננטה בשם `PasswordInput`.

הקומפוננטה תכלול:

* input לסיסמה
* כפתור שמחליף בין הצגת הסיסמה להסתרה
* טקסט קטן שמציג כמה תווים הוקלדו

דרישות:

* השתמשו ב־`useState` לשמירת ערך הסיסמה.
* השתמשו ב־`useState` נוסף כדי לדעת האם הסיסמה גלויה או מוסתרת.
* שנו את `type` של ה־input בין `"password"` ל־`"text"`.

# useEffect

## תרגיל 1 — קל: שינוי כותרת דף בטעינה

צרו קומפוננטה בשם `PageTitle`.

כאשר הקומפוננטה נטענת בפעם הראשונה, כותרת הדפדפן תשתנה ל:

```text
Welcome to React Hooks
```

דרישות:

* השתמשו ב־`useEffect`.
* ה־effect צריך לרוץ פעם אחת בלבד.

---

## תרגיל 2 — בינוני: מעקב אחרי חיפוש

צרו קומפוננטה בשם `SearchTracker`.

הקומפוננטה תציג input לחיפוש.

בכל פעם שהטקסט משתנה, הדפיסו לקונסול:

```text
Search changed: ...
```

דרישות:

* שמרו את ערך החיפוש ב־`useState`.
* השתמשו ב־`useEffect` שרץ רק כאשר ערך החיפוש משתנה.

רמז: מערך התלות צריך לכלול את המשתנה שמחזיק את הטקסט.

---

## תרגיל 3 — בינוני: הודעת שמירה אוטומטית

צרו קומפוננטה בשם `AutoSaveNote`.

הציגו textarea שבו המשתמש כותב הערה.

בכל פעם שהטקסט משתנה, הציגו הודעה:

```text
Saving...
```

ואחרי שנייה אחת החליפו אותה ל:

```text
Saved
```

דרישות:

* השתמשו ב־`useState` לטקסט.
* השתמשו ב־`useState` לסטטוס השמירה.
* השתמשו ב־`useEffect` שמגיב לשינוי בטקסט.

רמז: אפשר להשתמש ב־`setTimeout`. חשבו האם צריך לנקות timeout קודם.

# useRef

## תרגיל 1 — קל: ניקוי input בלחיצה

צרו קומפוננטה בשם `ClearInput`.

הקומפוננטה תציג input וכפתור:

```text
Clear
```

בלחיצה על הכפתור, תוכן ה־input יימחק.

דרישות:

* השתמשו ב־`useRef`.
* אין להשתמש ב־`document.querySelector`.

---

## תרגיל 2 — בינוני: בדיקת ערך קודם

צרו קומפוננטה בשם `PreviousName`.

הציגו input שבו המשתמש מקליד שם.

מתחת ל־input הציגו:

```text
Current name: ...
Previous name: ...
```

דרישות:

* השתמשו ב־`useState` לשם הנוכחי.
* השתמשו ב־`useRef` כדי לשמור את הערך הקודם.
* הערך הקודם צריך להתעדכן אחרי שינוי בשם.

רמז: שילוב של `useEffect` ו־`useRef` יכול לעזור כאן.

---

## תרגיל 3 — בינוני: מדידת גובה של אלמנט

צרו קומפוננטה בשם `BoxMeasure`.

הציגו קופסה עם טקסט בתוכה וכפתור:

```text
Measure Box
```

בלחיצה על הכפתור, הציגו את גובה הקופסה בפיקסלים.

דרישות:

* השתמשו ב־`useRef` כדי לגשת ל־div.
* השתמשו ב־`offsetHeight` או `getBoundingClientRect`.

רמז: ה־ref צריך להיות מחובר ל־div שאתן רוצות למדוד.

# useMemo

## תרגיל 1 — קל: חישוב ממוצע ציונים

צרו קומפוננטה בשם `AverageGrade`.

הגדירו מערך ציונים:

```jsx
const grades = [90, 80, 100, 70];
```

חשבו את הממוצע והציגו אותו במסך.

דרישות:

* השתמשו ב־`useMemo` לחישוב הממוצע.
* הוסיפו `console.log` בתוך החישוב כדי לראות מתי הוא רץ.

---

## תרגיל 2 — בינוני: סינון משימות לפי סטטוס

צרו קומפוננטה בשם `TaskFilter`.

הגדירו מערך משימות:

```jsx
const tasks = [
  { id: 1, title: "Learn hooks", done: true },
  { id: 2, title: "Practice React", done: false },
  { id: 3, title: "Build project", done: false }
];
```

הציגו שלושה כפתורי סינון:

```text
All
Done
Open
```

הציגו רק את המשימות שמתאימות לסינון הנבחר.

דרישות:

* השתמשו ב־`useState` לשמירת הסינון הנוכחי.
* השתמשו ב־`useMemo` לחישוב הרשימה המסוננת.

רמז: החישוב תלוי בערך הסינון.

## תרגיל 3 — בינוני: חישוב סל קניות

צרו קומפוננטה בשם `CartSummary`.

הגדירו מערך מוצרים בסל:

```jsx
const cartItems = [
  { id: 1, name: "Book", price: 50, quantity: 2 },
  { id: 2, name: "Pen", price: 5, quantity: 4 },
  { id: 3, name: "Notebook", price: 20, quantity: 1 }
];
```

חשבו:

* מחיר כולל
* כמות פריטים כוללת

דרישות:

* השתמשו ב־`useMemo` לחישוב הנתונים.
* הציגו את התוצאה במסך.
* הוסיפו state נוסף שלא קשור לחישוב, למשל `showDetails`, ובדקו האם החישוב רץ מחדש כשהוא משתנה.

רמז: אם החישוב לא תלוי ב־`showDetails`, הוא לא צריך להיות במערך התלות.

# useCallback

## תרגיל 1 — קל: פונקציית איפוס

צרו קומפוננטה בשם `ResetCounter`.

הציגו מונה וכפתורים:

```text
Increase
Reset
```

את פונקציית האיפוס הגדירו בעזרת `useCallback`.

דרישות:

* השתמשו ב־`useState` למונה.
* השתמשו ב־`useCallback` עבור פונקציית האיפוס.

---

## תרגיל 2 — בינוני: קומפוננטת ילד שמקבלת פעולה

צרו שתי קומפוננטות:

```text
ParentBox
ActionButton
```

`ParentBox` יחזיק state בשם `message`.

`ActionButton` יקבל פונקציה דרך props.

בלחיצה על הכפתור שב־`ActionButton`, ההודעה ב־`ParentBox` תשתנה ל:

```text
Action completed
```

דרישות:

* הגדירו את הפונקציה ב־`ParentBox` בעזרת `useCallback`.
* העבירו אותה ל־`ActionButton` כ־prop.

רמז: הילד לא צריך להחזיק state משלו.

---

## תרגיל 3 — בינוני: בדיקת רינדור של רכיב ילד

צרו קומפוננטה בשם `ParentWithInput`.

בתוכה יהיו:

* input לשם משתמש
* מונה
* קומפוננטת ילד בשם `LogButton`

`LogButton` תקבל פונקציה בשם `onLog`.

דרישות:

* עטפו את `LogButton` ב־`React.memo`.
* הוסיפו בתוך `LogButton` הדפסה לקונסול:

```jsx
console.log("LogButton rendered");
```

* הגדירו את `onLog` בעזרת `useCallback`.
* בדקו מה קורה כאשר מקלידים ב־input ומה קורה כאשר המונה משתנה.

רמז: אם הפונקציה משתמשת בערך מתוך state, צריך לחשוב אם הוא צריך להופיע במערך התלות.

# useContext

## תרגיל 1 — קל: צבע מועדף

צרו context בשם `FavoriteColorContext`.

ב־`App`, ספקו צבע מועדף, למשל:

```text
purple
```

צרו קומפוננטה בשם `FavoriteColorText` שמציגה:

```text
My favorite color is purple
```

דרישות:

* השתמשו ב־`createContext`.
* השתמשו ב־`Provider`.
* השתמשו ב־`useContext`.

---

## תרגיל 2 — בינוני: פרטי משתמש בכמה רכיבים

צרו context בשם `CurrentUserContext`.

הערך שיישמר בו יהיה אובייקט:

```jsx
{
  name: "Dana",
  role: "admin"
}
```

צרו שתי קומפוננטות:

```text
UserName
UserRole
```

`UserName` תציג את שם המשתמש.
`UserRole` תציג את התפקיד.

דרישות:

* שתי הקומפוננטות יקראו את אותו context.
* אין להעביר את הערכים דרך props.

---

## תרגיל 3 — בינוני: ניהול גודל טקסט גלובלי

צרו context בשם `FontSizeContext`.

האפליקציה תאפשר לבחור גודל טקסט גלובלי:

```text
small
medium
large
```

צרו שתי קומפוננטות:

```text
FontSizeControls
ArticleText
```

`FontSizeControls` תציג כפתורים לשינוי הגודל.
`ArticleText` תציג פסקה שהגודל שלה משתנה לפי הערך ב־context.

דרישות:

* ה־state של גודל הטקסט יהיה ב־`App`.
* הערך ופונקציית העדכון יעברו דרך context.
* אין להעביר props בין `App` לילדים עבור גודל הטקסט.

רמז: הערך של ה־Provider יכול להיות אובייקט שמכיל גם ערך וגם פונקציה.

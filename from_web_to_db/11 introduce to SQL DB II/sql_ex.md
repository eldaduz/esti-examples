

# 📘 SQL Practice – Lesson 11 (Aggregation & Grouping)

## 🎯 מטרת התרגול

בתרגול זה נלמד לעבוד עם:

* פונקציות צבירה (Aggregation)
* GROUP BY (קיבוץ נתונים)
* HAVING (סינון קבוצות)
* ההבדל בין WHERE ל־HAVING

נמשיך לעבוד עם הטבלה:

```text
employees
```

---

# 🧠 חלק א – פונקציות צבירה בסיסיות

## 🧩 תרגילים

1. כמה עובדים יש בטבלה?
2. כמה עובדים פעילים (is_active = true)?
3. מה השכר הממוצע של כל העובדים?
4. מה השכר הגבוה ביותר?
5. מה השכר הנמוך ביותר?
6. מה סכום כל המשכורות?

📌 שימו לב:

* כל השאילתות מחזירות שורה אחת בלבד
* השתמשו ב־COUNT / AVG / SUM / MIN / MAX

## 🧩 תרגילים

📌 רמז:

# 🧠 חלק ו – WHERE + GROUP BY

## 🧩 תרגילים

18. הציגו מספר עובדים בכל מחלקה — רק עובדים פעילים
19. הציגו שכר ממוצע לכל מחלקה — רק עובדים עם שכר מעל 15000
20. הציגו עובדים שנקלטו אחרי 2022 — ואז חשבו כמה יש בכל מחלקה

📌 חשוב:
WHERE מסנן שורות לפני GROUP BY

# 🧠 חלק ז – HAVING (סינון קבוצות)

## 🧩 תרגילים

21. הציגו מחלקות עם יותר מ־2 עובדים
22. הציגו מחלקות שבהן השכר הממוצע מעל 15000
23. הציגו מחלקות שבהן השכר המקסימלי מעל 20000

📌 רמז:

# 💡 טיפים חשובים

✔ WHERE → סינון שורות

✔ GROUP BY → קיבוץ

✔ HAVING → סינון קבוצות

✔ סדר עבודה:

```sql
FROM
WHERE
GROUP BY
HAVING
SELECT
ORDER BY
```
